    • prompt

    API

    该插件在安装后会在所有组件实例上提供 $prompt 对象，对象上提供如下方法：

    方法名类型描述
    successfunction(content, title, options): Promise<?boolean>

    该方法返回 Promise 对象，弹框关闭时 resolved。如果用户点击了“确定”按钮，resolve 结果值类型就为 string，表示获取的用户输入的文本内容；如果点击了“取消”按钮，结果值就为 null。

    参数名类型说明
    contentstring显示在输入弹框中的文本内容。
    titlestring输入弹框的标题。
    options!Object=见详情。
    名称类型描述
    openboolean是否显示输入弹框。
    overlayClassstring|Object|Array<string>添加到输入弹框浮层根节点的类名。
    okfunction(): Promise<void>点击“确定”按钮后，在 success 方法返回的 Promise resolve 前插入异步流程。如果该函数返回的 Promise 被 resolve，则弹框会被关闭，否则不会关闭。
    cancelfunction(): Promise<void>点击“取消”按钮后，在 success 方法返回的 Promise resolve 前插入异步流程。如果该函数返回的 Promise 被 resolve，则弹框会被关闭，否则不会关闭。
    warnfunction(content, title, options): Promise<?boolean>同 success。
    infofunction(content, title, options): Promise<?boolean>同 success。
    errorfunction(content, title, options): Promise<?boolean>同 success。
