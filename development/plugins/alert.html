    API

    该插件在安装后会在所有组件实例上提供 $alert 对象，对象上提供如下方法：

    方法名类型描述
    successfunction(content, title, options): Promise<void>

    该方法返回 Promise 对象，如果弹框成功关闭，则执行 resolved 流程，否则执行 rejected 流程。

    参数名类型说明
    contentstring显示在警告弹框中的文本内容。
    titlestring警告弹框的标题。
    options!Object=见详情。
    名称类型描述
    openboolean是否显示警告弹框。
    overlayClassstring|Object|Array<string>添加到警告弹框浮层根节点的类名。
    okfunction(): Promise<void>点击“知道了”按钮后调用。如果该函数返回的 Promise 进入 resolved 流程，则弹框会被关闭，否则不会关闭。
    warnfunction(content, title, options): Promise<void>同 success 。
    infofunction(content, title, options): Promise<void>同 success 。
    errorfunction(content, title, options): Promise<void>同 success 。
