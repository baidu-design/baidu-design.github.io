    • Toast 消息提示

    示例

    类型

    Toast 有四种类型，分别是 info、success、warning 和 error，可以通过 type 属性指定不同的类型。

    请填写完毕后继续。
    保存成功！
    请先登录后再进行提交。
    保存失败！
    <template>
<article>
  <section>
    <veui-toast
      type="info"
      open
      message="请填写完毕后继续。"
    />
  </section>
  <section>
    <veui-toast
      type="success"
      open
      message="保存成功！"
    />
  </section>
  <section>
    <veui-toast
      type="warning"
      open
      message="请先登录后再进行提交。"
    />
  </section>
  <section>
    <veui-toast
      type="error"
      open
      message="保存失败！"
    />
  </section>
</article>
</template>

<script>
import { Toast } from 'veui'

export default {
  components: {
    'veui-toast': Toast
  }
}
</script>

    命令式调用

    可使用 veui/plugins/toast 模块加载 toast 插件进行命令式调用。

    <template>
<article>
  <veui-button @click="$toast.info({
    message: '请填写完毕后继续。',
    duration: 5000
  })">信息提示</veui-button>
  <veui-button @click="$toast.success('保存成功！')">成功提示</veui-button>
  <veui-button @click="$toast.warn('请先登录后再进行提交。')">警告提示</veui-button>
  <veui-button @click="$toast.error('保存失败！')">错误提示</veui-button>
</article>
</template>

<script>
import Vue from 'vue'
import { Button } from 'veui'
import toast from 'veui/plugins/toast'

Vue.use(toast)

export default {
  components: {
    'veui-button': Button
  }
}
</script>

    API

    属性

    名称类型默认值描述
    openbooleanfalse

    .sync

    是否显示消息提示。

    typestring'success'

    警告框类型。

    值描述
    info信息提示样式。
    success成功样式。
    warning警告样式。
    error错误样式。
    messagestring-消息内容。
    durationnumber3000消息展示时间毫秒数，超过此事件则消息提示自动关闭。

    插槽

    名称描述
    default内容区。默认显示 message 属性的内容。

    事件

    名称描述
    close消息自动关闭后触发。

    全局配置

    配置项类型默认值描述
    toast.durationnumber3000消息默认展示时间毫秒数。

    图标

    名称描述
    info普通信息。
    success成功状态。
    warning警告状态。
    error错误状态。
    PromptBox - 输入弹框 Tooltip - 浮层提示
