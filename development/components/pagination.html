      <template>
<article>
  <veui-pagination :page="page" :total="total" :to="to" @redirect="handleRedirect"/>
  <veui-pagination :page="page" :total="total" :to="to" ui="hetero" @redirect="handleRedirect"/>
  <veui-pagination :page="page" :total="total" :to="to" ui="full" @redirect="handleRedirect"/>
  <veui-pagination :page="page" :total="total" :to="to" ui="slim" @redirect="handleRedirect"/>
  <veui-pagination :page="1" :total="0"/>
</article>
</template>

<script>
import { Pagination } from 'veui'

export default {
  components: {
    'veui-pagination': Pagination
  },
  data () {
    return {
      to: '/:page',
      page: 1,
      total: 10101
    }
  },
  methods: {
    handleRedirect ({page, event}) {
      this.page = page
    }
  }
}
</script>

      API

      属性

      名称类型默认值描述
      uistring-

      预设样式。

      值描述
      normal翻页按钮在两边
      hetero翻页按钮在右边
      full带条数和页数
      slim微型
      pagenumber1当前页。
      totalnumber-总个数。
      tostring|Object''

      目标链接模板。类型见 Link 组件的同名属性。其中，使用 string 路径时其中的 :page 关键词会被替换为实际页码。

      nativebooleanfalse原生链接跳转。
      page-sizenumberpagination.pageSize

      .sync

      每页个数。

      page-sizesArraypagination.pageSizes每页个数候选项。

      事件

      名称描述
      pagesizechangepage-size 改变时触发，回调参数为 (size: number) 。 size 为新的 page-size 值。
      redirect链接跳转时触发，回调参数为 (page: number, event: Object) 。 page 为要跳转的目标页码。 event 为原生的点击事件，调用 event.preventDefault 可阻止跳转。

      全局配置

      配置项类型默认值描述
      pagination.pageSizenumber30每页个数。
      pagination.pageSizesArray<number>[30, 50, 100]每页个数候选项。
