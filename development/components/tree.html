    <template>
<article>
  <div class="title">自定义文本区域：</div>

    <template slot="item-label" slot-scope="props">{{ props.item.value }} = {{ props.item.label }}</template>

  <div class="title">自定义整个节点区域：</div>
  <veui-tree :datasource="region" :expands="expands" item-click="toggle" class="custom-whole-item">

      <veui-icon
        v-if="props.item.children && props.item.children.length"
        :name="props.item.expanded ? 'minus-small' : 'plus-circle-o'"
      ></veui-icon>
      <span>{{ props.item.value }} = {{ props.item.label }}</span>
    </template>
  </veui-tree>

</template>

    API

    属性

    名称类型默认值描述
    datasourceArray[]数据源，数据项类型为 {value: *, label: string, ...}。
    item-clickstringnone

    点击整个节点区域进行的操作类型。

    名称描述
    toggle触发展开/收起。
    none无操作。
    expandsArray[]指定哪些节点是展开的，是一个对应于 datasource 中 value 属性的数组。

    作用域插槽

    名称描述
    item

    每个节点的整个内容区域。

    名称类型描述
    itemObject当前数据节点。
    indexnumber当前数据节点在共父节点层级的索引。
    depthnumber当前数据节点在树种的深度。
    item-label每个节点的文本区域。参数属性与 item 的一致。

    事件

    名称描述
    click

    点击节点触发，回调参数为 (item, parents, index, depth)。

    名称类型描述
    itemObject节点数据。
    parentsArray<Object>从根节点到当前节点父节点的路径。
    indexnumber当前节点在自己所属层级的索引。
    depthnumber当前节点在树形层级中的深度。
    expand

    点击展开图标或者整个节点时触发，由 item-click 属性决定，回调参数为 (item, index, depth)。

    名称类型描述
    itemObject节点数据。
    indexnumber当前节点在自己所属层级的索引。
    depthnumber当前节点在树形层级中的深度。
    collapse

    点击收起图标或者整个节点时触发，由 item-click 属性决定，回调参数为 (item, index, depth)。

    名称类型描述
    itemObject节点数据。
    indexnumber当前节点在自己所属层级的索引。
    depthnumber当前节点在树形层级中的深度。

    图标

    名称描述
    expand收起状态，点击后展开。
    collapse展开状态，点击后收起。
