    API

    属性

    名称类型默认值描述
    uistring-

    预设样式。

    值描述
    small小尺寸样式。
    tiny特小尺寸样式。
    value*|Array<*>-

    值。

    默认值类型为 number，值大小范围应在 min 和 max 之间。 当值为 Array<number> 形式时，组件则渲染多个滑块与值一一对应。

    当提供 parse 和 format 时，值可以为任意类型，但 parse 需要把传入值解析为 number 类型并返回，format 需要把传出值格式化成原始形式。parse 和 format 的实现只需要处理单值，组件内部会给多值的每一项分别依次调用这两个函数。

    secondary-progressnumber|Array<number>0次要条。
    minnumber0最小值。
    maxnumber1最大值。
    stepnumber0步进值。
    markbooleanfalse是否显示步进标记。
    parsefunctionval => val处理传入值。
    formatfunctionval => val处理传出值。

    事件

    名称描述
    input

    v-model

    修改后触发，回调参数为 (value: *)。value 为改变后的新值。

    插槽

    名称描述
    track滑轨。默认内容：横线。
    tip-label浮动提示文本。默认内容：当前 value 值。

    作用域插槽

    名称描述
    thumb

    滑块。

    默认内容：圆形。

    名称类型描述
    indexnumber滑块索引。
    focusboolean滑块是否聚焦。
    hoverboolean鼠标是否悬浮。
    draggingboolean滑块是否正在被拖动。
    tip

    浮动提示。

    默认内容：内容为 value 的 <veui-tooltip>

    名称类型描述
    targetHTMLElement滑块元素。
    openboolean是否显示浮动提示。
    active-indexboolean激活的滑块索引。
