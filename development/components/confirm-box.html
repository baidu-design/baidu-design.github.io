    • ConfirmBox 确认弹框

    示例

    自定义标题和内容区。

    系统提示

    删除后不可恢复，确认删除？

    <template>
<article>
  <veui-button @click="open = true">删除</veui-button>
  <veui-confirm-box
    title="系统提示"
    :open.sync="open"
    @ok="ok"
    @cancel="cancel"
  >
    <p>删除后不可恢复，确认删除？</p>
  </veui-confirm-box>
</article>
</template>

<script>
import { ConfirmBox, Button } from 'veui'
import toastManager from 'veui/managers/toast'

export default {
  components: {
    'veui-confirm-box': ConfirmBox,
    'veui-button': Button
  },
  data () {
    return {
      open: false
    }
  },
  methods: {
    ok () {
      this.open = false
      toastManager.info('已确认')
    },
    cancel () {
      toastManager.info('已取消')
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    openbooleanfalse

    .sync

    是否显示确认弹框。

    titlestring-标题。
    overlay-classstring|Array|Object-参考 Overlay 组件的 overlay-class 属性。

    插槽

    名称描述
    default内容区。
    title标题区。若同时指定了 title 属性和 title 插槽，以后者为准。
    foot底部区域，默认会展示“确定”、“取消”按钮。

    事件

    名称描述
    ok点击“确定”按钮时触发。
    cancel点击“取消”按钮时触发。
    AlertBox - 警告弹框PromptBox - 输入弹框
