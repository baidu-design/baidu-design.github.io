    • BreadcrumbItem 面包屑项

    BreadcrumbItem 组件需要在 Breadcrumb 组件内使用。

    示例

    见 Breadcrumb 示例。

    API

    属性

    名称类型默认值描述
    tostring|Object-点击面包屑项之后要跳转到的地址。见 Link 组件的同名属性。
    typestring'link'

    面包屑项的类型。

    值描述
    link超链接样式。
    text文本样式。
    nativebooleanfalse当值为 true 且指定了 to 时，会直接使用原生 <a> 元素，to 值会被设置到 <a> 标签的 href 属性上。

    插槽

    名称描述
    default填充面包屑项的正文内容区。

    事件

    名称描述
    redirect点击 type 为 link 的项目时将触发该事件，回调参数为 (event: Event)，直接透传原生点击事件对象。

    图标

    名称描述
    separator分隔符。
    :focus-visibleveui-loader
