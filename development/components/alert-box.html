    示例

    类型

    AlertBox 有三种类型，分别是 info、success 和 error，可以通过 type 属性指定不同的类型。

    成功啦！

    订单创建成功

    失败了！

    订单创建失败

    提示

    广告已下线，请知晓
    <template>
<article>
  <veui-button @click="successOpen = true">成功</veui-button>
  <veui-button @click="errorOpen = true">错误</veui-button>
  <veui-button @click="infoOpen = true">信息</veui-button>

  <veui-alert-box
    type="success"
    :open.sync="successOpen"
  >
    <template slot="title">成功啦！</template>
    <div>订单创建成功</div>
  </veui-alert-box>

  <veui-alert-box
    type="error"
    :open.sync="errorOpen"
  >
    <template slot="title"></template>
    <div>订单创建失败</div>
  </veui-alert-box>

  <veui-alert-box
    type="info"
    :open.sync="infoOpen"
  >
    <template slot="title"></template>
    <div>广告已下线，请知晓</div>
  </veui-alert-box>
</article>
</template>

<script>
import AlertBox from 'veui/components/AlertBox'
import Button from 'veui/components/Button'

export default {
  components: {
    'veui-alert-box': AlertBox,
    'veui-button': Button
  },
  data () {
    return {
      successOpen: false,
      errorOpen: false,
      infoOpen: false
    }
  }
}
</script>

    标题

    可以通过 title 属性或 title 插槽自定义警告弹框的标题。

    来自系统的提示

    订单创建成功

    成功啦

    订单创建成功

    <template>
<article>
  <veui-button @click="open1 = true">通过属性自定义标题</veui-button>
  <veui-button @click="open2 = true">通过插槽自定义标题</veui-button>

  <veui-alert-box
    ui="success"
    :open.sync="open1"
    title="来自系统的提示"
    @ok="ok"
  >
    <div>订单创建成功</div>
  </veui-alert-box>

  <veui-alert-box
    ui="success"
    :open.sync="open2"
    @ok="ok"
  >
    <template slot="title">成功啦 <veui-icon name="info-circle" class="icon"/></template>
    <p>订单创建成功</p>
  </veui-alert-box>
</article>
</template>

<script>
import AlertBox from 'veui/components/AlertBox'
import Button from 'veui/components/Button'
import Icon from 'veui/components/Icon'
import 'veui-theme-one-icons/info-circle'
import toastManager from 'veui/managers/toast'

export default {
  components: {
    'veui-alert-box': AlertBox,
    'veui-button': Button,
    'veui-icon': Icon
  },
  data () {
    return {
      open1: false,
      open2: false
    }
  },
  methods: {
    ok () {
      toastManager.info('已确认')
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    openbooleanfalse

    .sync

    是否显示警告弹框。

    typestring'success'

    警告框类型。

    值描述
    info信息提示样式。
    success成功样式。
    error错误样式。
    titlestring-标题。
    overlay-classstring|Array|Object-参考 Overlay 组件的 overlay-class 属性。

    插槽

    名称描述
    default内容区。
    title标题区。若同时指定了 title 属性和 title 插槽，以后者为准。
    foot底部区域，默认会展示一个“知道了”按钮。

    事件

    名称描述
    ok点击“知道了”按钮触发。
    afterclose浮层关闭后触发。如果样式主题提供了退出动画，将在退出动画完毕后触发。

    图标

    名称描述
    info普通信息。
    success成功状态。
    error错误状态。
    Steps - 步骤条ConfirmBox - 确认弹框
