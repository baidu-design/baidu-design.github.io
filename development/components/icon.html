    <template>
<article>
  <div class="item" v-for="icon in icons" :key="icon">
    <div class="icon"><veui-icon :name="icon"/></div>
    <div class="name">{{ icon }}</div>
  </div>
</article>
</template>

<script>
import { Icon } from 'veui'
import 'veui-theme-one-icons'

export default {
  components: {
    'veui-icon': Icon
  },
  data () {
    return {
      icons: Object.keys(Icon.icons).sort((a, b) => a > b ? 1 : -1)
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    namestring-图标名称。
    labelstring-图标的文字说明，对辅助设备可见。当不设置时，图标对辅助设备隐藏。
    scalenumber-图标尺寸倍数。不设置时相当于 1。
    spinbooleanfalse是否保持旋转状态。
    pulsebooleanfalse是否保持步进旋转状态。
    inversebooleanfalse是否翻转颜色（用白色绘制，用于深色背景）。
    flipstring-是否翻转，可选值为 'horizontal'/'vertical'，分别表示水平翻转与垂直翻转。

    spin、pulse 属性会占用图标根元素的 CSS animation 属性，flip 则会占用其 transform 属性，如果需要设置其它 animation 或 transform 属性。请避免在这些属性同时使用 animation/transform 以防止互相覆盖，需要对应效果时应自行提供完整的 animation/transform 声明。

    插槽

    名称描述
    default用来实现图标堆叠，当内部有 Icon 子组件时会层叠在一起居中显示，且外部 Icon 组件不需要设置 name 属性。
