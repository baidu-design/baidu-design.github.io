    示例

    单日选择

    默认情况下，单击日期可以选中一个日期。

    一二三四五六日

    已选日期：2018 M08 1, Wed

    支持 v-model，数据类型为原生 Date 类型。 

    <template>
<article>
  <veui-calendar v-model="date"/>
  <p>已选日期：{{ readableDate }}</p>
</article>
</template>

<script>
import { Calendar } from 'veui'

export default {
  components: {
    'veui-calendar': Calendar
  },
  data () {
    return {
      date: new Date()
    }
  },
  computed: {
    readableDate () {
      return this.date.toLocaleDateString('zh-CN', {
        weekday: 'long',
        year: 'numeric',
        month: 'long',
        day: 'numeric'
      })
    }
  }
}
</script>

    多日选择、日期范围选择

    配置 multiple 属性时，可以选择多个日期。配置 range 属性时，可以选择一个日期范围。

    多日选择

    一二三四五六日
    已选日期：待选择

    日期范围选择

    一二三四五六日
    已选日期：待选择

    支持 v-model，选择多个单日时数据类型为 Array<Date>，选择日期范围时数据类型为 [Date, Date]。 

    <template>
<article>
  <section class="col">
    <h4>多日选择</h4>
    <veui-calendar v-model="dates" multiple/>
    <section>已选日期：{{ readableDates }}</section>
  </section>
  <section class="col">
    <h4>日期范围选择</h4>
    <veui-calendar v-model="range" range/>
    <section>已选日期：{{ readableRange }}</section>
  </section>
</article>
</template>

<script>
import { Calendar } from 'veui'

export default {
  components: {
    'veui-calendar': Calendar
  },
  data () {
    return {
      dates: null,
      range: null
    }
  },
  computed: {
    readableDates () {
      if (!this.dates || !this.dates.length) {
        return '待选择'
      }
      return this.toReadable(this.dates).join('、')
    },
    readableRange () {
      if (!this.range) {
        return '待选择'
      }
      return this.toReadable(this.range).join(' 至 ')
    }
  },
  methods: {
    toReadable (dates) {
      return dates.map(date =>
        date.toLocaleDateString('zh-CN')
      )
    }
  }
}
</script>

    多日期范围选择

    同时配置 multiple 及 range 属性时，可以选择多段日期范围。配置 panel 属性时，可以指定日历面板的数量。两次选择的时间段合并的策略为异或逻辑。

    一二三四五六日
    一二三四五六日
    一二三四五六日
    已选日期：待选择

    支持 v-model，选择多日期范围时数据类型为 Array<[Date, Date]>。 

    <template>
<article>
  <veui-calendar v-model="ranges" range multiple :panel="3"/>
  <section>已选日期：{{ readableRanges }}</section>
</article>
</template>

<script>
import { Calendar } from 'veui'

export default {
  components: {
    'veui-calendar': Calendar
  },
  data () {
    return {
      ranges: null
    }
  },
  computed: {
    readableRanges () {
      if (!this.ranges) {
        return '待选择'
      }
      return this.ranges.map(range => this.toReadable(range).join(' 至 ')).join('、')
    }
  },
  methods: {
    toReadable (dates) {
      return dates.map(date =>
        date.toLocaleDateString('zh-CN')
      )
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    multiplebooleanfalse是否可以选择多个日期（范围）。
    rangebooleanfalse是否选择日期范围。
    selectedDate|Array-

    v-model

    已选日期（范围）的值，根据 multiple 和 range 属性值的不同，数据格式不同。

    multiplerange类型
    falsefalseDate
    truefalseArray<Date>
    falsetrue[Date, Date]
    truetrueArray<[Date, Date]>
    panelnumber1日历面板数量。
    todayDatenew Date()「今天」的日期。
    week-startnumbercalendar.weekStart一周的起始。可进行 全局配置 。
    fill-monthbooleantrue当只有一个面板时，是否要在当前月份面板显示非本月日期。
    date-classstring|Array|Object|function{}特定日期的自定义 HTML class 。传非函数时，数据格式为所有 Vue 支持的 class 表达式 ；传函数时，签名为 function(Date): string|Array<string>|Object<string, boolean> ，返回值格式亦为所有 Vue 支持的 class 表达式。
    disabled-datefunction(Date): boolean() => false特定日期是否禁用。
    disabledbooleanfalse是否为禁用状态。
    readonlybooleanfalse是否为只读状态。

    插槽

    名称描述
    before日历内，面板上方可供定制的区域。
    after日历内，面板下方可供定制的区域。

    作用域插槽

    名称描述
    date

    单日单元格内的区域，可用来定制每一天对应区域的内容。

    默认内容：对应日期的 date。

    名称类型描述
    yearnumber完整年份。
    monthnumber月份数， 0 表示一月。
    datenumber月份内的日期。

    事件

    名称描述
    select

    v-model

    选择修改后触发，回调参数为 (selected)。数据类型和 selected 属性一致。

    selectstart选择日期范围时，选择完起始日期时触发，回调参数 (picking: Date) ，表示已选的起始项日期。
    selectprogress

    选择日期范围时，在已经选择开始日期后，通过鼠标或键盘交互标记到的结束日期变更时触发。回调参数为 (picking)，表示如果以当前标记的日期为中止日期，日历组件最终将输出的 selected 值，类型取决于 multiple 属性的值。

    multiple类型
    false[Date, Date]
    trueArray<[Date, Date]>
    viewchange第一个面板显示的月份发生变化时触发。回调参数 (month: Object<{year: number, month: number}>) ，表示当前年月（ month 为 0 表示一月）。

    全局配置

    名称类型默认值描述
    calendar.weekStartnumber1一周的第一天是星期几。周一到周日分别对应 1 到 7 。
