    <template>
<article>
  <veui-filterpanel :datasource="region" :search-on-input="false">
    <template slot-scope="scope">
      <div
        v-for="item in scope.items"
        :key="item.value"
        v-show="!item.hidden"
        class="item"
      >{{ item.label }}</div>
    </template>
  </veui-filterpanel>
</article>
</template>

<script>
import { FilterPanel } from 'veui'
import region from '@/common/region'

export default {
  components: {
    'veui-filterpanel': FilterPanel
  },
  data () {
    return {
      region: region[0].children[0].children[0].children
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    datasourceArray<Object>[]数据源，项目类型为 {label: string, value: *}。
    searchablebooleantrue是否允许搜索。
    filterfunction见描述

    搜索过滤函数，签名为 function(keyword, option, index, datasource): boolean。返回值为 false 的项目将被从结果中过滤掉。

    名称类型描述
    keywordstring搜索关键词。
    optionObject当前遍历到的数据节点。
    indexnumber当前数据节点在兄弟节点中的索引。
    datasourceArray<{label: string, value: *}>与 datasource 属性一致。
    import { includes } from 'lodash'

function (keyword, item) {
  return includes(item.label, keyword)
}
    search-on-inputbooleantrue是否在输入的时候触发搜索。
    placeholderstring-搜索框的占位符。

    插槽

    名称描述
    head标题区域。
    no-datadatasource 中没数据时显示的内容。

    作用域插槽

    名称描述
    default

    主内容区域。

    名称类型描述
    itemsArraydatasource 经过过滤后的数据，类型与 datasource 保持一致。
