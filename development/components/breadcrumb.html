    • Breadcrumb 面包屑

    Breadcrumb 组件可以内联 BreadcrumbItem 组件使用。

    示例

    直接内联 BreadcrumbItem 使用。

    1. 百度首页
    2. 百度知道
    3. 爱奇艺
    <template>
<article>
  <veui-breadcrumb>
    <veui-breadcrumb-item native to="https://www.baidu.com">百度首页</veui-breadcrumb-item>
    <veui-breadcrumb-item native to="https://zhidao.baidu.com">百度知道</veui-breadcrumb-item>
    <veui-breadcrumb-item type="text">爱奇艺</veui-breadcrumb-item>
  </veui-breadcrumb>
</article>
</template>

<script>
import { Breadcrumb, BreadcrumbItem } from 'veui'

export default {
  components: {
    'veui-breadcrumb': Breadcrumb,
    'veui-breadcrumb-item': BreadcrumbItem
  }
}
</script>

    也可以传入数据源。

    1. 按钮
    2. 面包屑
    3. 日期选择
    <template>
<article>
  <veui-breadcrumb :routes="routes"/>
</article>
</template>

<script>
import { Breadcrumb, BreadcrumbItem } from 'veui'

export default {
  components: {
    'veui-breadcrumb': Breadcrumb,
    'veui-breadcrumb-item': BreadcrumbItem
  },
  data () {
    return {
      routes: [
        {
          to: '/development/components/button',
          text: '按钮'
        },
        {
          to: '/development/components/breadcrumb',
          text: '面包屑'
        },
        {
          text: '日期选择'
        }
      ]
    }
  }
}
</script>

    监听跳转事件。

    1. 按钮
    2. 面包屑
    3. 日期选择
    Redirect to -.
    <template>
<article>
  <veui-breadcrumb :routes="routes" @redirect="handleRedirect"/>
  <div class="console">Redirect to {{ currentRoute.text }}.</div>
</article>
</template>

<script>
import { Breadcrumb, BreadcrumbItem, Icon } from 'veui'
import 'veui-theme-one/icons/arrow-right'

export default {
  components: {
    'veui-breadcrumb': Breadcrumb,
    'veui-breadcrumb-item': BreadcrumbItem,
    'veui-icon': Icon
  },
  data () {
    return {
      currentRoute: {
        text: '-'
      },
      routes: [
        {
          text: '按钮',
          type: 'link'
        },
        {
          text: '面包屑',
          type: 'link'
        },
        {
          text: '日期选择'
        }
      ]
    }
  },
  methods: {
    handleRedirect (event, route) {
      this.currentRoute = route
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    routesArray<Object>[]

    面包屑配置，项目类型为 {label: string, type: string, to: string|Object, native: boolean=}，除 label 外字段详情可参考 BreadcrumbItem 组件的属性。

    默认情况下，最后一项始终会显示成普通文本样式。

    插槽

    名称描述
    default支持内联模式，直接传入 BreadcrumbItem 组件列表。填充后 routes 属性将被忽略。

    作用域插槽

    名称描述
    default

    用于为每个项目自定义内容。默认内容为 routes 项目的 label 或 BreadcrumbItem 组件的默认插槽。

    名称类型描述
    routeObjectroutes 中的项目。自定义字段亦会绑定到作用域对象中。
    separator面包屑条目之间的分割符。如果不传，则使用全局配置的分割图标；如果传入，则用指定模板插槽填充分割区域。

    事件

    名称描述
    redirect

    点击每个条目都会触发该事件，回调参数为 (event, route, index)。

    名称类型描述
    eventEvent事件对象。
    routeObject面包屑条目对象。
    indexnumber当前点击条目的索引。
