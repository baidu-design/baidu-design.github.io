    • Dialog 对话框

    API

    模态对话框与非模态对话框。

    <template>
<article>
  <veui-button @click="modalVisible = true">模态</veui-button>
  <veui-button @click="modelessVisible = true">非模态</veui-button>
  <veui-dialog
    :open.sync="modalVisible"
    @ok="modalVisible = false"
    @cancel="modalVisible = false"/>
  <veui-dialog
    :open.sync="modelessVisible"
    :modal="false"
    @ok="modelessVisible = false"
    @cancel="modelessVisible = false"/>
</article>
</template>

<script>
import { Dialog, Button } from 'veui'

export default {
  components: {
    'veui-dialog': Dialog,
    'veui-button': Button
  },
  data () {
    return {
      modalVisible: false,
      modelessVisible: false
    }
  }
}
</script>

    标题

    这里是自定义的内容区域。

    <template>
<article>
  <veui-button @click="simpleVisible = true">简单自定义</veui-button>
  <veui-dialog
    :open.sync="simpleVisible"
    title="这里是自定义的标题"
    @ok="simpleVisible = false"
    @cancel="simpleVisible = false"
  >这里是自定义的内容区域。</veui-dialog>

  <veui-button @click="titleIconVisible = true">标题中插入图标</veui-button>
  <veui-dialog
    :open.sync="titleIconVisible"
    @ok="titleIconVisible = false"
    @cancel="titleIconVisible = false"
  ><template slot="title">标题<veui-icon name="top"/></template>这里是自定义的内容区域。</veui-dialog>

  <veui-button @click="footVisible = true">自定义底部按钮</veui-button>
  <veui-dialog :open.sync="footVisible">
    <template slot="foot">
      <veui-button ui="small primary" @click="footVisible = false">关闭</veui-button>
    </template>
  </veui-dialog>
</article>
</template>

<script>
import { Dialog, Button, Icon } from 'veui'
import 'veui-theme-one/icons/top'

export default {
  components: {
    'veui-dialog': Dialog,
    'veui-button': Button,
    'veui-icon': Icon
  },
  data () {
    return {
      simpleVisible: false,
      titleIconVisible: false,
      footVisible: false
    }
  }
}
</script>

    属性

    名称类型默认值描述
    uistring-

    绑在浮层根元素上的 ui 属性。

    值描述
    top在水平方向居中，垂直方向靠上取黄金分割点。
    modalbooleantrue是否是模态对话框。模态对话框默认遮挡底层（无法点击）且抢占焦点（关闭后焦点会回归）。
    titlestring-对话框标题文本。如果指定了 title 插槽，则优先使用 title 插槽。
    openbooleanfalse

    .sync

    是否显示对话框。

    closablebooleantrue是否显示关闭按钮。
    draggablebooleanfalse是否可拖拽。
    escapablebooleanfalse按下 esc 键是否可以关闭对话框。仅在 closable 为 true 时生效。
    prioritynumber-对话框浮层层叠权重，参考 Overlay 组件 。
    before-closefunction(string): boolean=|Promise<boolean=>-

    在将触发对话框关闭的操作发生后执行，参数列表为 (type: string)。type 为触发关闭的类型，默认情况下可选值为 'ok'|'cancel'。返回值可以是一个 boolean，也可以是一个 resolve boolean 的 Promise，用来处理可能需要异步决定对话框关闭状态的情况。返回值或 resolve 值非 false 时才会关闭对话框。

    <veui-dialog :open.sync="dialogOpen" :before-close="submit">...</veui-dialog>
    methods: {
  submit (type) {
    if (type === 'ok') {
      return axios.post('/item/create', {/* ... */})
        .then(({ id, error }) => {
          if (error) {
            this.showError(error)
            return false // resolve `false` 将阻止对话框关闭
          }
        })
    }
    // resolve 但不返回 `false` 时会关闭对话框
  },
  // ...
}
    overlay-classstring|Object-对话框浮层根元素类名，参考 Overlay 组件 。

    插槽

    名称描述
    default内容区。
    title标题区。若同时指定了 title 属性和 title 插槽，以后者为准。
    foot底部区域，默认会展示“确定”、“取消”按钮。

    事件

    名称描述
    ok点击“确定”按钮时触发。
    cancel点击“取消”按钮时触发。

    图标

    名称描述
    close关闭。
