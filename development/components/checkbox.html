    • Checkbox 复选框

    示例

    尺寸

    可选的 ui 属性值：small。

    使用 indeterminate 属性切换复选框是否处于部分选中的状态。

    <template>
<article>
  <section>
    <veui-checkbox :indeterminate.sync="indeterminate" v-model="normal">中尺寸</veui-checkbox>
  </section>
  <veui-checkbox v-model="indeterminate" ui="small">部分选择状态</veui-checkbox>
</article>
</template>

<script>
import { Checkbox } from 'veui'

export default {
  components: {
    'veui-checkbox': Checkbox
  },
  data () {
    return {
      normal: false,
      indeterminate: false
    }
  }
}
</script>

    值设定

    可以通过设置 true-value 和 false-value 来修改选中、非选中状态下 model 属性（v-model）的值。

    状态：待确认

    <template>
<article>
  <veui-checkbox v-model="status" true-value="CONFIRMED" false-value="UNCONFIRMED">我已阅读用户协议</veui-checkbox>
  <p>状态：{{ statusMap[status] }}</p>
</article>
</template>

<script>
import { Checkbox } from 'veui'

const STATUS_MAP = {
  CONFIRMED: '已确认',
  UNCONFIRMED: '待确认'
}

export default {
  components: {
    'veui-checkbox': Checkbox
  },
  data () {
    return {
      status: 'UNCONFIRMED',
      statusMap: STATUS_MAP
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    uistring-

    预设样式。

    值描述
    small小尺寸样式。
    model*-

    v-model

    当前状态的值。选中状态为 true-value 对应值，未选中状态为 false-value 对应值。

    checkedbooleanfalse

    .sync

    是否处于选中状态。

    true-value*true选中状态对应的值。
    false-value*false未选状态对应的值。
    indeterminatebooleanfalse

    .sync

    是否处于半选状态。

    disabledbooleanfalse是否为禁用状态。
    readonlybooleanfalse是否为只读状态。

    插槽

    名称描述
    default复选框的描述文本，点击时会切换选择状态。无默认内容。

    事件

    名称描述
    change用户切换选中状态时触发，回调参数为 (checked: boolean)。checked 表示当前是否选中。
    input

    v-model

    选中状态变化后触发，回调参数为 (val: *)。val 为当前 model 属性的值。与 change 事件不同，input 事件在数据操作导致状态变化时也会触发。

    此外，Checkbox 组件透传了如下的原生事件：

    click、keyup、keydown、keypress、focus、blur。

    回调参数均为相应的原生事件对象。

    图标

    名称描述
    checked已选状态。
    indeterminate半选状态。
    RadioButtonGroup - 单选按钮组 CheckboxGroup - 复选框组
