    • RadioGroup 单选框组

    示例

    尺寸

    可选的 ui 属性值：small。

    普通尺寸

    小尺寸

    <template>
<article>
  <section>
    <h4>普通尺寸</h4>
    <veui-radio-group :items="licenses"/>
  </section>
  <section>
    <h4>小尺寸</h4>
    <veui-radio-group ui="small" :items="licenses"/>
  </section>
</article>
</template>

<script>
import { RadioGroup } from 'veui'

export default {
  components: {
    'veui-radio-group': RadioGroup
  },
  data () {
    return {
      licenses: [
        {
          label: 'MIT License',
          value: 'mit'
        },
        {
          label: 'BSD License',
          value: 'bsd'
        },
        {
          label: 'Apache License 2.0',
          value: 'apache2'
        }
      ]
    }
  }
}
</script>

    API

    属性

    名称类型默认值描述
    uistring-

    预设样式。

    值描述
    small小尺寸样式。
    itemsArray<Object>[]

    单选框组数据源，项目类型为 {label, value, disabled, ...}。

    名称类型描述
    labelstring选项的文字说明。
    value*选项对应的值。
    disabledboolean=选项是否为禁用。
    value*-

    v-model

    items 中已选项的 value。

    disabledbooleanfalse是否为禁用状态。
    readonlybooleanfalse是否为只读状态。

    作用域插槽

    名称描述
    default

    选项描述文本区域。

    默认内容：label 属性值。

    名称类型描述
    labelstring选项文本。
    valuestring选项值。
    indexnumber选项在 items 中的序号。
    disabledboolean选项是否禁用。

    另外，items 内数据项中除了上面描述的字段之外的其它字段也会自动通过 v-bind 进行绑定到作用域参数上。

    事件

    名称描述
    change

    v-model

    选中状态变化后触发，回调参数为 (value: *)。value 为当前按钮组已选项内 value 字段的值。
